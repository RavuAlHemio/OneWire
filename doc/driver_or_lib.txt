This file documents whether a 1-Wire Public Domain Kit function is a driver function (different for each method of
access) or a library function (shared by all drivers).

D  driver function (lib/)
L  library function (common/)
S  driver function (lib/) whose implementation is shared by all drivers (move to library)
C  driver function (lib/) whose implementation is shared by all drivers but called by at least one driver function
   (don't move to library)
